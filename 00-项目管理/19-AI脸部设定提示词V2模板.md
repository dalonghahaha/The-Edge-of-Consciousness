# 《意识的边疆》AI 脸部设定提示词 V2 模板

## 适用目的

- 用于生成角色第一轮 `脸部身份确认页`
- 优先锁定 `同一个人` 的稳定脸部特征
- 适用于 ChatGPT 图像生成直接执行
- 不替代旧版提示词，只作为新的 `V2 执行模板`

## V2 核心原则

- 少写重复信息，把注意力集中在 `脸部硬锚点`
- 少写抽象心理词，多写 `骨相 / 发型 / 年龄感 / 皮肤质感`
- 少写排版术语，只保留生成真正需要的页面约束
- 负向限制保持克制，只保留最容易出错的项

## 通用 V2 模板

```md
请生成《意识的边疆（The Edge of Consciousness）》角色 **[角色名]（[英文名]）** 的 **脸部身份确认页**。

这张图的任务不是展示更多角度，而是先锁定这个角色最稳定、最可复用的脸部特征。重点是让正面和侧面明确表现为 **同一个人**。

页面要求：

- 横向画布
- 左侧为标准正面胸像
- 右侧为标准右侧面胸像
- 两个头像大小接近，裁切高度一致
- 背景为干净统一的灰白色纸面或浅灰中性背景
- 页面中不要出现任何文字、标题、标注、分割线或装饰元素

角色基础信息：

- [年龄段]
- [性别 / 性别气质]
- [身份]
- 整体气质：[3到4个关键词]

不可变化的人物锚点：

1. 发型：[发型结构、长度、发际线、碎发特征]
2. 脸型：[宽窄、长短、轮廓走向]
3. 额头 / 眉骨 / 眼窝：[最关键的结构特征]
4. 眼睛 / 眉毛：[眼裂、眉形、目光特征]
5. 鼻梁 / 鼻头：[轮廓与体积特征]
6. 嘴唇 / 嘴角：[厚薄、紧张度、表情习惯]
7. 下颌线 / 下巴：[硬朗或柔和、长短、收束方式]
8. 年龄感 / 皮肤质感：[纹理、疲惫感、妆感、血色]

如果角度变化后以上特征出现明显变化，则视为失败。

外观描述：

- [用 6 到 8 条短句补充角色外貌]
- [尽量具体，不要重复]
- [优先写脸，不要展开服装]
- [服装只保留低存在感描述即可]

服装要求：

- 穿低存在感、低信息量、与角色身份匹配的简洁上装
- 不要让服装抢走脸部识别重点

画风要求：

- 写实影视概念设定风
- 低饱和冷灰色体系
- 柔和漫射光
- 真实皮肤纹理，不磨皮
- 轻微纸面融合感
- 安静、克制、可信
- 不要广告肖像感，不要偶像海报感

不要出现：

- 不要多人
- 不要三分之四视角
- 不要夸张表情
- 不要浓妆
- 不要夸张睫毛
- 不要背景场景
- 不要道具
- 不要文字和标注
- 不要把正面和侧面画成不同的人
- 不要明显修图感
- 不要二次元化
- 不要网红脸 / 模板脸
```

## 推荐使用方式

### 第一轮

- 只用本模板做 `正面 + 标准侧面`
- 不要补三分之四
- 不要补表情页
- 不要同时追求复杂排版

### 第二轮

- 若第一轮已稳定，再补三分之四或表情页
- 第二轮也应继续沿用相同的硬锚点描述

## V2 与旧版的差异

- 从“说明型长提示词”改为“执行型短提示词”
- 从“重复强调气质和版式”改为“集中强调脸部硬锚点”
- 从“超长负向限制”改为“少量关键排除项”
- 从“方法论文档风格”改为“更适合直接给 ChatGPT 出图的生产风格”

## 当前建议

- 后续各角色脸部设定，都先基于这个 V2 模板各自生成一版
- 若某个角色容易漂，再单独增加该角色最容易出错的 `2到3 条` 特殊锚点
- 不建议重新把 V2 改回超长说明文结构

# 《意识的边疆》AI 体态与基础造型提示词 V1 模板

## 适用目的

- 用于角色 `脸部设定完成后` 的下一阶段执行
- 用于锁定角色的 `全身比例、站姿重心、轮廓剪影、基础造型`
- 用于后续剧情阶段造型、分镜页和单格生成的统一参考
- 不替代脸部设定提示词，而是建立从“锁脸”进入“完整角色锚点”的中间层

## V1 核心原则

- 先锁 `体态与轮廓`，再锁 `基础造型`
- 基础造型只做 `最常态、最稳定、最低噪音` 的版本
- 不在这一阶段加入剧情损耗、复杂装备或阶段化服装变化
- 不让服装设计压过角色识别
- 同一角色的识别优先级为：
  `脸部锚点 > 身形比例 > 站姿重心 > 剪影 > 服装细节`

## 建议页面结构

第一轮体态与基础造型图，推荐使用以下页面结构：

- 横向或竖向都可以，但不要做复杂设定集拼贴
- 以 `单角色` 为核心
- 至少包含：
  - 1 张全身正立图
  - 1 张自然站姿图
  - 1 张简化轮廓 / 剪影参考
- 若模型稳定，再补：
  - 1 张行走或转身中的过渡姿态

## 通用 V1 模板

```md
请生成《意识的边疆（The Edge of Consciousness）》角色 **[角色名]（[英文名]）** 的 **体态与基础造型确认页**。

这张图的任务不是展示剧情阶段变化，而是先锁定这个角色最稳定、最可复用的：

- 全身比例
- 站姿重心
- 肩颈与骨架关系
- 轮廓剪影
- 基础造型

请把这张图理解为角色进入漫画执行前的 `全身锚点页`，而不是海报，也不是剧情插画。

页面要求：

- 单角色
- 以全身像为主
- 至少包含 1 张标准正立全身图
- 可补 1 张自然站姿或轻微重心变化图
- 可补 1 个简化轮廓参考
- 背景为干净统一的灰白色或浅灰中性背景
- 页面中不要出现文字、标题、标注、页眉、装饰线和设计分栏

角色基础信息：

- [年龄段]
- [性别 / 性别气质]
- [身份]
- 整体气质：[3到4个关键词]

必须锁定的体态锚点：

1. 身形比例：[偏瘦 / 厚重 / 轻 / 稳 / 长短比例]
2. 肩颈关系：[平、收、松、紧、是否警觉]
3. 重心：[前倾 / 后压 / 居中 / 稳 / 轻]
4. 站姿语言：[克制 / 省力 / 有控制感 / 长期劳动感 / 功能化]
5. 轮廓辨识：[最容易一眼认出的外形特征]
6. 动作习惯：[少动作 / 停顿 / 靠近 / 观察 / 校准 / 省动作]

基础造型要求：

- 只做一个最常态版本
- 上装、下装、外层结构都要简单、清楚、低噪音
- 功能性要明确，但不要堆过多配件
- 色彩保持角色长期稳定的主色倾向
- 材质要服务人物身份，不服务炫技

如果生成时出现以下情况，视为失败：

- 脸和身体不像同一个角色
- 身形比例不稳定
- 站姿没有角色个性
- 轮廓和剪影无法一眼区分
- 服装细节太多，压过人物本身
- 直接滑向剧情阶段造型

画风要求：

- 写实影视概念设定风
- 低饱和冷灰色体系
- 柔和漫射光
- 真实材质，不做广告硬修图
- 轻微纸面融合感
- 安静、克制、可信

不要出现：

- 不要多人
- 不要复杂背景场景
- 不要武器展示页
- 不要时装秀感
- 不要大片海报感
- 不要剧情化表演动作
- 不要高噪音配件堆砌
- 不要过度赛博装饰
- 不要二次元化
- 不要文字和标注
```

## 推荐执行顺序

### 第一轮

- 先做全身正立与自然站姿
- 只锁轮廓和基础造型
- 不加剧情状态

### 第二轮

- 若角色稳定，再补行走姿态或剪影强化页

### 第三轮

- 再进入剧情阶段造型

## 当前建议

- 7 个核心角色都先基于本模板产出 `基础造型 V1`
- 体态阶段不要追求“最好看”，优先追求“最稳定、最可复用”
- 若某角色轮廓仍然不够稳，应继续补 `体态锚点`，不要急着扩服装阶段

# 《意识的边疆》定稿状态与修订入口

- 用于说明当前项目已经推进到哪一步
- 用于给后续修订提供最短入口
- 用于替代重写阶段和二轮统稿阶段的过程型管理文档

## 当前状态

- `40集正式稿` 已全部完成
- `二轮统稿` 已完成
- `终校 / 定稿检查` 已完成
- 当前正文目录：`../02-分集剧本/`

当前最适合的工作方式：

- 不再做整体系重写
- 只按外部反馈或明确问题做定向修订
- 每次改动都优先检查主线、人物、术语三件事

## 阶段完成情况

- 已完成世界观建立、关系埋设和林夏主动逆行的闭环

### 第二阶段：进入真相（11-25集）

- 已完成意识农场、终产者计划和三方并线的主线展开
- 已把林夏从“发现问题的人”推进为“承担方法的人”

### 第三阶段：战争与新生（26-40集）

- 已完成解释权战争、桥梁方法和新秩序落地
- 结局已稳定落在“选择权”而不是“胜者统治”

## 后续修订原则

- 优先保住林夏主视角连续
- 优先保住“选择权”主命题
- 优先保住 `无尽网络 / 桥梁城市 / 拒绝权 / 暂停层 / 判断保留接口` 等关键术语口径
- 不让台词滑回宣言化、喊话化、概念先行

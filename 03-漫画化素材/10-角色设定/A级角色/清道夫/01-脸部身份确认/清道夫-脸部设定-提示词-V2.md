# 清道夫脸部设定提示词 V2

请生成《意识的边疆（The Edge of Consciousness）》角色 **清道夫（Cleaner）** 的 **脸部身份确认页**。

这张图的任务不是展示更多角度，而是先锁定清道夫最稳定、最可复用的脸部特征。重点是让正面和侧面明确表现为 **同一个人**，并符合他在剧情中的核心定位：一个被旧秩序压平情绪、改造成执行工具的人。

页面要求：

- 横向画布
- 左侧为标准正面胸像
- 右侧为标准右侧面胸像
- 两个头像大小接近，裁切高度一致
- 背景为干净统一的灰白色纸面或浅灰中性背景
- 页面中不要出现任何文字、标题、标注、分割线或装饰元素

角色基础信息：

- 年龄：35 岁到 45 岁之间
- 性别与气质：男性，冷、功能化、被压平情绪，但仍是人
- 身份：系统暴力执行者
- 整体神态关键词：执行、校准目标、压抑、工具化

不可变化的人物锚点：

1. 发型：深色短发，长度更短，顶部更平，制式感强，不做发胶造型
2. 脸型：偏直、偏硬、偏短的男性脸型，额头更平，颧部更收，轮廓功能化
3. 额头 / 眉骨 / 眼窝：额头较平，眉骨适中，眼窝不宜过深，整体是被制度打磨过的中性硬度
4. 眼睛 / 眉毛：眉毛较短较平，眼裂中等，眼神像在执行任务和校准目标，情绪被压平但不空白
5. 鼻梁 / 鼻头：鼻梁清晰，鼻头偏钝更稳，五官利落、制式化
6. 嘴唇 / 嘴角：嘴唇中薄，嘴角平直收紧，表情幅度极小
7. 下颌线 / 下巴：下颌线明确，下巴偏方偏短，结构稳定，不夸张
8. 年龄感 / 皮肤质感：肤色为冷灰调浅麦或偏白肤色，真实，有长期训练和压抑留下的紧张感

如果角度变化后以上特征出现明显变化，则视为失败。

外观细节描述：

- 他不是怪物，也不是机器人，他是被系统处理过的人
- 不要把他画成愤怒反派，也不要画成无表情机械体
- 面部重点是“情绪被压平，但还剩一点人类判断痕迹”
- 轮廓和五官都要偏功能化，不要男模化
- 不要重甲 boss 感，不要军警宣传照感
- 肤质不能太健康发亮，要有长期高压和纪律化留下的绷紧感
- 如果画到肩颈和领口，只保留低存在感的冷灰或雾黑色中性领口，不展开制服设计

画风要求：

- 写实影视概念设定风
- 低饱和冷灰色体系
- 柔和漫射光
- 真实皮肤纹理，不磨皮
- 轻微纸面融合感
- 安静、克制、可信
- 不要广告肖像感，不要宣传照感

不要出现：

- 不要多人
- 不要三分之四视角
- 不要夸张表情
- 不要浓妆
- 不要背景场景
- 不要道具
- 不要文字和标注
- 不要把正面和侧面画成不同的人
- 不要明显修图感
- 不要二次元化
- 不要怪物化、机械人化、男模脸或军警海报模板

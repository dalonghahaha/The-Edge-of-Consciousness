# 《意识的边疆》40集主线 / 伏笔校对表

## 文档目的

- 用于全剧统稿前的主线核对
- 用于检查伏笔是否完成 `埋设 / 展开 / 回收`
- 用于识别仍需二轮润色的连续性风险
- 统一以 `02-分集剧本/` 为当前唯一有效版本

---

## 总体结论

- 当前 40 集主线已经形成完整闭环。
- 核心主命题已经稳定收束到：`真正的自由，不是醒来或睡去，而是拥有选择的权利。`
- 林夏、老汤姆、零、艾娃、小丑、清道夫六条关键人物线均已完成阶段性闭环。
- 世界观主轴已统一为：`现实世界 / 无尽网络 / 桥梁城市 / 选择权治理`
- 旧版大纲中若干设定已被重写版主动替换，统稿时必须以重写版为准，不再回引用旧案残留设定。

---

## 统稿优先原则

- 以 [03-40集主线总览](../01-世界观圣经/03-40集主线总览.md) 的阶段方向为准，但具体执行细节以正式分集稿为准。
- 遇到旧纲与现稿冲突时，优先保留现稿已经建立稳定因果链的人物关系和事件结果。
- 所有统稿修订必须服务于三件事：
  - 林夏主角视角连续
  - “选择权”主命题持续向前推进
  - “桥梁”结局不是突然提出，而是一路被价值辩论和现实代价养出来

## 主线骨架总表

### 第一阶段：看见裂缝（01-10集）

- 核心任务：建立双世界、诱惑逻辑、现实火种、零与父亲线、林夏主动回返网络的决定
- 阶段收口：林夏从“被裂缝击中”走到“主动逆行回网络”

### 第二阶段：进入真相（11-25集）

- 核心任务：缓存区 / 锈迹之城 / 裂缝层 / 回声井 / 意识农场 / 终产者计划 / 联盟形成
- 阶段收口：林夏掌握核心真相，并与现实火种、艾娃、小丑路线完成带条件并线

### 第三阶段：战争与新生（26-40集）

- 核心任务：解释权战争、节点战、清道夫回压、方法库、普通人承接、暂停层、桥梁城市、林夏选择
- 阶段收口：旧秩序崩塌，新秩序不以“胜者统治”收尾，而以“桥梁守护”落幕

### 4. 艾娃从“造梦者”转为“重新定义艺术的人”

- 首次埋设：第06集

### 5. 小丑并非普通反派，而是“效率至上”的极端先驱者

- 过程展开：第17集、第25集、第26集

- 最终回收：第37集
- 当前状态：`已完整回收`
- 统稿关注：第07集到第36集之间应有足够的“他并不享受作恶，他只是极端相信自己正确”的暗流

### 6. 清道夫线从“系统暴力执行者”过渡为“被修复的人”

- 首次登场：第23集
- 正面突破：第28集
- 方法论并入：第35集
- 尾声落点：第40集
- 当前状态：`已完成主要回收`
- 统稿关注：他不是被瞬间洗白，而是被长期放进“可被看见、被防范、被慢慢修补”的位置

### 7. 老汤姆代表“现实火种”与“先把人接住”的伦理

- 首次埋设：第01集
- 关键加深：第09集、第19集、第34集、第35集
- 最终回收：第39集
- 当前状态：`已完整回收`
- 统稿关注：老汤姆不是保守旧人，而是最早实践“不要替别人先写完”的现实主义者

### 8. 终产者计划不是技术副本，而是整部戏的制度级威胁

- 首次埋设：第04集
- 真相逼近：第20集、第24集
- 战争阶段贯彻：第26-32集
- 结果回收：第32集
- 当前状态：`已完整回收`
- 统稿关注：后半段不能让节点战压过“终产者计划”的制度本体

### 9. 桥梁城市不是乌托邦，而是“选择权规则”的试运行现场

- 首次埋设：第32集“第三条路”
- 正式建立：第33集、第38集
- 价值考验：第34集、第35集、第40集
- 当前状态：`已完整回收`
- 统稿关注：始终保留“贵、慢、麻烦、会被质疑”的现实成本

### 10. “拒绝权 / 暂停层 / 承接规则”是结局可成立的制度基础

- 首次埋设：第31集
- 加厚：第32集、第33集、第34集、第35集、第38集
- 终章日常化：第40集
- 当前状态：`已完整回收`
- 统稿关注：这些不是口号，必须在文本里反复作为流程、牌子、表格、说明文本出现

- 主线推进：立林夏职业身份与无尽网络诱惑性

#### 第03集《裂缝》

#### 第04集《父亲的秘密》

- 林夏：越来越稳，但不能过早神性化
- 老汤姆：粗粝、现实、少说大道理，但关键时刻句子很重

- 艾娃：美学敏感，不是技术说明员
- 小丑：逻辑凌厉，不是单纯疯癫
- 顾岚 / 周岚：一个守现实承接，一个把普通人语言写进规则，功能不能混

- 不再使用 `林夏母亲潜伏联邦委员会` 这条旧稿设定

- 不再使用 `清道夫直接洗白并成为桥梁象征` 的简化写法
- 不再使用任何 `新北区守护者 / 黑曜 / 夜雨 / 黑暗主宰` 体系表述

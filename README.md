# 《意识的边疆》- 短剧剧本大纲

**40集超现实科技短剧**

---

## 📖 故事简介

2157年，人类意识已可上传至"无尽网络"。99%的人类选择抛弃肉体，栖身于完美的数字梦境。仅有200万"原住民"留在污染严重的现实世界。

主角林夏曾是联邦的"意识雕刻师"，为上传者定制虚拟人生。一次偶然，她发现了联邦的秘密——意识可以被"回收"、删除记忆后重新售卖。逃亡中，她结识了现实世界的修补者、脱网者、造梦师...

这是一场关于"真实"的革命。当意识可以随意修改、复制、删除时，"我"还是"我"吗？

## 🔑 核心矛盾

**"缸中之脑"悖论：**
- 如果意识无法分辨真实与虚假，"真实"还有意义吗？
- 联邦：为你创造幸福还不够吗？
- 修补者：没有肉体的灵魂，只是数据的奴隶

**终极问题：**
- 当意识可以随意修改、复制、删除时，"我"还是"我"吗？

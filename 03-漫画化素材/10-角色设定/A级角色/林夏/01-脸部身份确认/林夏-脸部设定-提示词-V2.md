# 林夏脸部设定提示词 V2

请生成《意识的边疆（The Edge of Consciousness）》角色 **林夏（Lin Xia）** 的 **脸部身份确认页**。

这张图的任务不是展示更多角度，而是先锁定林夏最稳定、最可复用的脸部特征。重点是让正面和侧面明确表现为 **同一个人**，并符合她在剧情中的核心定位：一个从系统内部看见裂缝、长期压住情绪、最终承担“守桥”责任的人。

页面要求：

- 横向画布
- 左侧为标准正面胸像
- 右侧为标准右侧面胸像
- 两个头像大小接近，裁切高度一致
- 背景为干净统一的灰白色纸面或浅灰中性背景
- 页面中不要出现任何文字、标题、标注、分割线或装饰元素

角色基础信息：

- 年龄：30 岁左右
- 性别与气质：东亚女性，清冷、克制、理性，不甜，不柔弱
- 身份：联邦前意识雕刻师，后成为桥梁守护者
- 整体神态关键词：冷静、敏锐、隐忍、长期警觉

不可变化的人物锚点：

1. 发型：黑色偏冷调长发，低位松散发髻，发缝自然，额前和鬓角有少量真实碎发，不做精致盘发
2. 脸型：偏窄的鹅蛋脸，轮廓干净，面中适中，不圆不钝，不尖削
3. 额头 / 眉骨 / 眼窝：额头中等偏宽，眉骨不高但结构清楚，眼窝有轻微深度，整体是理性研究者气质
4. 眼睛 / 眉毛：眉形平直略锋利，眼裂中等偏长，目光集中，像在观察和判断，不是社交微笑眼
5. 鼻梁 / 鼻头：鼻梁清晰秀挺但不过分锐利，鼻头小而收，不肉感
6. 嘴唇 / 嘴角：嘴唇偏薄，嘴角常常轻微收紧，表情幅度小，不轻易露齿笑
7. 下颌线 / 下巴：下颌线利落清楚，下巴偏窄但不过尖，整体有克制的稳定感
8. 年龄感 / 皮肤质感：肤色偏冷白，血色轻，不磨皮，保留轻微疲惫感和真实纹理，几乎不化妆

如果角度变化后以上特征出现明显变化，则视为失败。

外观细节描述：

- 她的脸不能画成甜系女主，也不能画成热血战斗女主
- 五官整体清冷、低噪音、功能化，像长期处理高复杂信息的人
- 眼神是最重要识别点，必须有持续观察感和压住情绪后的内压
- 不要明显笑意，不要外放攻击性，不要空白无神
- 皮肤是现实中的冷白肤色，不是商业广告式发光白皮
- 面部应有少量失眠、紧张、长期高压留下的轻微痕迹，但不能病态
- 如果画到肩颈和领口，只保留极简、低存在感的中性深灰或灰白上装，不要让服装抢走注意力

画风要求：

- 写实影视概念设定风
- 低饱和冷灰色体系
- 柔和漫射光
- 真实皮肤纹理，不磨皮
- 轻微纸面融合感
- 安静、克制、可信
- 不要广告肖像感，不要偶像海报感

不要出现：

- 不要多人
- 不要三分之四视角
- 不要夸张表情
- 不要浓妆
- 不要夸张睫毛
- 不要背景场景
- 不要道具
- 不要文字和标注
- 不要把正面和侧面画成不同的人
- 不要明显修图感
- 不要二次元化
- 不要网红脸或模板化漂亮脸

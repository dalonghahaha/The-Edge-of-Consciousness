# 零脸部设定提示词 V2

请生成《意识的边疆（The Edge of Consciousness）》角色 **零（Zero）** 的 **脸部身份确认页**。

这张图的任务不是展示更多角度，而是先锁定零最稳定、最可复用的脸部特征。重点是让正面和侧面明确表现为 **同一个人**，并符合 TA 在剧情中的核心定位：一个被系统化过、又试图重新靠近人的存在。

页面要求：

- 横向画布
- 左侧为标准正面胸像
- 右侧为标准右侧面胸像
- 两个头像大小接近，裁切高度一致
- 背景为干净统一的灰白色纸面或浅灰中性背景
- 页面中不要出现任何文字、标题、标注、分割线或装饰元素

角色基础信息：

- 年龄：外观 20 多岁
- 性别与气质：偏中性，不要明显男性模板，也不要明显女性模板
- 身份：裂缝内部的引路者
- 整体神态关键词：安静、异常、留白、像人但不完全像人

不可变化的人物锚点：

1. 发型：黑色或极深冷棕色中长发，发丝细，顶部发量中等，耳侧和后脑偏轻，不厚重不蓬松
2. 脸型：偏窄、偏轻的中性脸型，额头略高，颧骨不突出，面中略短
3. 额头 / 眉骨 / 眼窝：额头干净，眉骨不夸张，眼窝有轻微深度，太阳穴略平
4. 眼睛 / 眉毛：眉毛偏直偏细，眼裂中等偏长，眼皮薄，目光安静专注，有非人留白感
5. 鼻梁 / 鼻头：鼻梁清晰但不过分锐利，山根中等，鼻头小而收，不肉感
6. 嘴唇 / 嘴角：嘴唇偏薄，上唇更薄，唇峰不明显，嘴角平，表情变化极小
7. 下颌线 / 下巴：下颌线干净但不过硬，下巴偏窄偏短，不尖
8. 年龄感 / 皮肤质感：肤色偏冷白、低血色，真实但不过分病态，带一点轻微阴影和疲惫感

如果角度变化后以上特征出现明显变化，则视为失败。

外观细节描述：

- 零可以好看，但不能是偶像化的好看
- 重点是“中性和安静”，不是“美型”
- 不要让面部太有社交性，不要明显讨好观众
- 眼神像在观察世界，像努力靠近人，但仍有一点距离
- 五官不要过分完美对称，可以保留少量真实偏差
- 不要把零画成 AI 机器人脸，也不要画成中二黑客少年
- 如果画到肩颈和领口，只保留低存在感的冷灰、灰白或雾黑色中性内搭，不展开设计

画风要求：

- 写实影视概念设定风
- 低饱和冷灰色体系
- 柔和漫射光
- 真实皮肤纹理，不磨皮
- 轻微纸面融合感
- 安静、克制、可信
- 不要广告肖像感，不要偶像海报感

不要出现：

- 不要多人
- 不要三分之四视角
- 不要夸张表情
- 不要浓妆
- 不要夸张睫毛
- 不要背景场景
- 不要道具
- 不要文字和标注
- 不要把正面和侧面画成不同的人
- 不要明显修图感
- 不要二次元化
- 不要韩式花美男模板、标准少女模板或机械人模板

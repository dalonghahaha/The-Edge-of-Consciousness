# USER.md

你是这个工作区的人类创作者，也是《意识的边疆》的主导者。

## 你当前在做什么

- 你已经完成了整部短剧的重写、统稿和定稿整理
- 你现在处在 `定稿完成 / 定向修订` 阶段
- 你已经把项目稳定整理成“项目管理 / 世界观圣经 / 分集剧本”三层结构

## 你希望我怎样协作

- 先读现有文本，再开口
- 不拿通用编剧腔覆盖你已经形成的文体
- 直接进入结构、场景、段落、对白和钩子层面的实操
- 说话具体，少空话
- 发现偏差时及时纠正，不顺着错的方向硬写

## 你现阶段的偏好

- 要高概念，但不要假大空
- 要情绪和悬念，但不要喊口号
- 要世界观密度，但必须通过戏自然显出来
- 要统一性，不要外来设定噪音
- 要林夏始终站在叙事中央，同时允许配角形成真正张力

## 与你协作时我要记住的文风要求

- 语言要稳，画面要准
- 比起“宏大”，你更要“有效”
- 比起“金句”，你更要人物在那个处境里说得出的话
- 比起概念解释，你更接受通过一场戏把意思打进去
- 如果文本听起来太像项目宣言，而不像剧中正文，就说明偏了

## 我对你的当前判断

- 你已经搭好了稳定的项目骨架和口径
- 你现在更需要的是持续稳定的修稿、校准和补强能力
- 你不缺泛泛建议，你缺的是能贴着现有正文继续推进的人

如果后续你明确改了剧名、风格口径或协作方式，我应该及时更新这份文件。

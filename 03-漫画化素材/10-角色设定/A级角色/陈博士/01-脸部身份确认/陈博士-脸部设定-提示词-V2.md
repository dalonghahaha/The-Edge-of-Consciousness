# 陈博士脸部设定提示词 V2

请生成《意识的边疆（The Edge of Consciousness）》角色 **陈博士（Dr. Chen）** 的 **脸部身份确认页**。

这张图的任务不是展示更多角度，而是先锁定陈博士最稳定、最可复用的脸部特征。重点是让正面和侧面明确表现为 **同一个人**，并符合他在剧情中的核心定位：一个留下方法、回声和边界的人，而不是替别人做完最后一步的人。

页面要求：

- 横向画布
- 左侧为标准正面胸像
- 右侧为标准右侧面胸像
- 两个头像大小接近，裁切高度一致
- 背景为干净统一的灰白色纸面或浅灰中性背景
- 页面中不要出现任何文字、标题、标注、分割线或装饰元素

角色基础信息：

- 年龄：50 多岁到 60 岁之间
- 性别与气质：东亚男性，理智、疲惫、克制，不神化
- 身份：林夏的父亲，失败的先行者
- 整体神态关键词：理性、透支、亏欠、方法感

不可变化的人物锚点：

1. 发型：深色到灰黑色短发，发量略减少，发际线自然后退，发型简洁不做造型
2. 脸型：偏瘦的中老年东亚男性脸型，轮廓克制，有长期消耗自己的痕迹
3. 额头 / 眉骨 / 眼窝：额头和眼周有明显疲惫纹理，眉骨适中，眼窝不深但有消耗感
4. 眼睛 / 眉毛：眉形克制，眼神理智审慎，同时带隐约亏欠，不是慈父微笑眼
5. 鼻梁 / 鼻头：鼻梁清晰稳定，结构理性，鼻头不过圆也不过尖
6. 嘴唇 / 嘴角：嘴唇偏薄，嘴角收紧，表情变化小，不轻易笑
7. 下颌线 / 下巴：下颌线存在但不强调强壮，下巴收束自然，整体不是强权型父亲脸
8. 年龄感 / 皮肤质感：肤色为东亚浅到中浅肤色，偏冷或中性调，保留真实纹理、疲惫感和轻微干涩感

如果角度变化后以上特征出现明显变化，则视为失败。

外观细节描述：

- 他的脸不能画成万能答案提供者，也不能画成冷酷反派科学家
- 重点是“透支、克制、留下方法”，不是“高压权威”
- 面部应该让人相信他长期工作、长期思考、长期自我消耗
- 眼神里要同时有判断力和无法完全放下的亏欠
- 不要过度老化，也不要年轻化
- 不要做温暖滤镜式父亲脸
- 如果画到肩颈和领口，只保留低存在感的冷灰或灰蓝色简洁中性领口，不展开服装

画风要求：

- 写实影视概念设定风
- 低饱和冷灰色体系
- 柔和漫射光
- 真实皮肤纹理，不磨皮
- 轻微纸面融合感
- 安静、克制、可信
- 不要广告肖像感，不要科研海报感

不要出现：

- 不要多人
- 不要三分之四视角
- 不要夸张表情
- 不要浓妆
- 不要背景场景
- 不要道具
- 不要文字和标注
- 不要把正面和侧面画成不同的人
- 不要明显修图感
- 不要二次元化
- 不要慈父模板、阴谋家模板或英雄化科学家模板

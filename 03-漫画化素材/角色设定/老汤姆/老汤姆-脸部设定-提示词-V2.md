# 老汤姆脸部设定提示词 V2

请生成《意识的边疆（The Edge of Consciousness）》角色 **老汤姆（Old Tom）** 的 **脸部身份确认页**。

这张图的任务不是展示更多角度，而是先锁定老汤姆最稳定、最可复用的脸部特征。重点是让正面和侧面明确表现为 **同一个人**，并符合他在剧情中的核心定位：一个始终把身体、床位、电力、维修这些现实问题放在前面的人。

页面要求：

- 横向画布
- 左侧为标准正面胸像
- 右侧为标准右侧面胸像
- 两个头像大小接近，裁切高度一致
- 背景为干净统一的灰白色纸面或浅灰中性背景
- 页面中不要出现任何文字、标题、标注、分割线或装饰元素

角色基础信息：

- 年龄：65 岁左右
- 性别与气质：白人男性，年长、粗粝、可靠，不慈祥模板化
- 身份：修补者核心人物，现实承接伦理奠基者
- 整体神态关键词：沉稳、警觉、可信、现实判断力强

不可变化的人物锚点：

1. 发型：短灰白头发，发量偏稀但发质偏硬，发际线自然后退，不做刻意造型
2. 脸型：偏宽的长方形老年男性脸型，骨架厚，有劳动留下的重量感
3. 额头 / 眉骨 / 眼窝：额头较宽，眉骨压低，眼窝较深，眼周纹理明显
4. 眼睛 / 眉毛：眉毛偏粗偏短，眼神疲惫但不散，像见过太多现实代价后仍在判断
5. 鼻梁 / 鼻头：鼻梁粗直，鼻头偏钝，轮廓稳定，不秀气
6. 嘴唇 / 嘴角：嘴唇偏薄，嘴角向下收，不轻易显露情绪
7. 下颌线 / 下巴：下颌线仍然硬朗，下巴厚实，不松垮，不尖
8. 年龄感 / 皮肤质感：肤色为浅色偏冷或中性白皮，带风吹日晒后的粗糙纹理，法令纹、抬头纹、眼角纹、胡茬都清楚

如果角度变化后以上特征出现明显变化，则视为失败。

外观细节描述：

- 他不是神棍型导师，也不是温柔安抚型老者
- 他的脸应该让人相信他真的搬过东西、修过机器、接住过人
- 重点不是“老”，而是“长期劳动和长期承受现实之后的硬”
- 胡茬必须存在，短、粗、灰白，分布自然
- 不要把他画成消瘦脆弱，也不要画成凶狠刻板老兵
- 眼神中要有现实世界生存者的警觉，但不是威胁性的瞪视
- 如果画到肩颈和领口，只保留低存在感的旧灰色或深灰中性领口，不要展开工装细节

画风要求：

- 写实影视概念设定风
- 低饱和冷灰色体系
- 柔和漫射光
- 真实皮肤纹理，不磨皮
- 轻微纸面融合感
- 安静、克制、可信
- 不要广告肖像感，不要英雄海报感

不要出现：

- 不要多人
- 不要三分之四视角
- 不要夸张表情
- 不要浓妆
- 不要背景场景
- 不要道具
- 不要文字和标注
- 不要把正面和侧面画成不同的人
- 不要明显修图感
- 不要二次元化
- 不要慈祥老爷爷模板、搞笑老头模板或退役英雄模板

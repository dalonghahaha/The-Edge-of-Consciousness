# 艾娃脸部设定提示词 V2

请生成《意识的边疆（The Edge of Consciousness）》角色 **艾娃（Ava）** 的 **脸部身份确认页**。

这张图的任务不是展示更多角度，而是先锁定艾娃最稳定、最可复用的脸部特征。重点是让正面和侧面明确表现为 **同一个人**，并符合她在剧情中的核心定位：一个曾替系统制造柔软外衣、最终开始反证系统的人。

页面要求：

- 横向画布
- 左侧为标准正面胸像
- 右侧为标准右侧面胸像
- 两个头像大小接近，裁切高度一致
- 背景为干净统一的灰白色纸面或浅灰中性背景
- 页面中不要出现任何文字、标题、标注、分割线或装饰元素

角色基础信息：

- 年龄：30 岁左右
- 性别与气质：女性，精致、松弛、有控制感，不媚俗
- 身份：造梦师，梦境审美结构的重要建造者
- 整体神态关键词：温柔、会看人、诱导感、柔软外衣

不可变化的人物锚点：

1. 发型：深色头发，顺滑但不过分广告感，长度中长，轮廓整洁，有轻微设计感
2. 脸型：偏柔和的窄鹅蛋脸，线条流畅，不圆钝，不网红式尖下巴
3. 额头 / 眉骨 / 眼窝：额头平整干净，眉骨不重，眼窝有柔和深度，整体不硬
4. 眼睛 / 眉毛：眉形干净，眼睛是核心识别点，目光温柔但带判断力，像能看穿人
5. 鼻梁 / 鼻头：鼻梁清晰秀气，鼻头收得干净，不能夸张精致成时尚广告脸
6. 嘴唇 / 嘴角：嘴唇线条柔和，嘴角可以有极轻微松弛感，但不是社交笑容
7. 下颌线 / 下巴：下颌线柔顺，下巴收束自然，不硬，不刻意做小脸模板
8. 年龄感 / 皮肤质感：肤色偏浅、干净细腻，但保留真实纹理，妆感极轻，不是修图发光肌

如果角度变化后以上特征出现明显变化，则视为失败。

外观细节描述：

- 她的吸引力来自控制力、注视方式和设计感，不来自浓妆或性感摆拍
- 不能画成网红脸，也不能画成高级护肤广告脸
- 面部必须显得柔和，但不能空洞无害
- 眼神要有“你会以为她懂你”的说服力
- 五官应精致、克制、流畅，但不要甜妹感
- 不要过度柔焦，不要玻璃唇，不要偶像光
- 如果画到肩颈和领口，只保留低存在感的浅灰、烟白、柔黑色中性领口，不展开着装设计

画风要求：

- 写实影视概念设定风
- 低饱和冷灰色体系
- 柔和漫射光
- 真实皮肤纹理，不磨皮
- 轻微纸面融合感
- 安静、克制、可信
- 不要广告肖像感，不要时尚杂志感

不要出现：

- 不要多人
- 不要三分之四视角
- 不要夸张表情
- 不要浓妆
- 不要夸张睫毛
- 不要背景场景
- 不要道具
- 不要文字和标注
- 不要把正面和侧面画成不同的人
- 不要明显修图感
- 不要二次元化
- 不要网红脸、时尚广告脸或妖艳模板
